# Colca Artesanías 🏔️

Sitio web profesional para plataforma de exportación artesanal del **Valle del Colca**, Arequipa, Perú. Proyecto basado en el Lienzo de Propuesta de Valor con datos de PROMPERÚ, MINCETUR y Perú Moda Deco 2025.

## Vista previa

| Página | Descripción |
|--------|-------------|
| `index.html` | Landing principal con hero, propuesta de valor, categorías, artesanos y mercados |
| `pages/catalogo.html` | Catálogo con filtros por categoría y grid de productos |
| `pages/artesanos.html` | Perfiles detallados de los artesanos con métricas |
| `pages/contacto.html` | Formulario de contacto con información del proyecto |

## Stack

- **HTML5** semántico y accesible
- **CSS3 puro** — Variables CSS, Grid, Flexbox, animaciones
- **JavaScript vanilla** — Sin frameworks, sin dependencias
- **Google Fonts** — Cormorant Garamond + Outfit
- **Intersection Observer API** — Animaciones scroll eficientes
